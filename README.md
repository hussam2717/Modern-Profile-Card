# 🎨 Card Design Project

## 📖 Description
This project is a modern and responsive card UI design built using HTML and CSS.  
It focuses on clean layout, styling, and responsive design principles.

## 🚀 Features
- Responsive card design
- Clean UI layout
- Hover effects
- Mobile-friendly design
- Simple and reusable structure

## 🛠️ Tech Stack
- HTML5
- CSS3

## 📚 What I Learned
- How to structure HTML properly
- CSS styling and layout techniques
- Flexbox for alignment
- Responsive design basics
- Hover and transition effects
